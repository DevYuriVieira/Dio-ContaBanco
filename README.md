## 🇬🇧 English

### 📌 About the Project
This repository contains a Java console application developed as a hands-on challenge for the **Digital Innovation One (DIO)**.

The goal is to simulate the creation of a bank account by capturing user input via terminal, processing primitive data types using the `Scanner` class, and handling basic input exceptions.

### ⚙️ Features
- Standardized terminal data entry
- Data validation using `try-catch` (`InputMismatchException`)
- Clean output formatting with concatenated variables
- Simple and intuitive CLI interaction

### 💻 Technologies
- **Java 17**
- **IDE:** IntelliJ IDEA (or any Java IDE / Terminal)

### 🚀 How to Run
1. Clone the repository:  
   git clone https://github.com/DevYuriVieira/Dio-ContaBanco.git

2. Navigate to the src directory:  
   cd Dio-ContaBanco/src

3. Compile the application:  
   javac ContaTerminal.java

4. Run the application:  
   java ContaTerminal

### 🖥️ Example Output
Por favor, digite o número da Agência:  
067-8  
Por favor, digite o número da Conta:  
1021  
Por favor, digite o nome do Cliente:  
Mario Andrade  
Por favor, digite o Saldo:  
237.48

Olá Mario Andrade, sua agência é 067-8, conta 1021 e seu saldo 237.48 já está disponível para saque
